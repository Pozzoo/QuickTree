# QuickTree

**Cut down entire trees in one swift motion with  physics-based animations!**

QuickTree is a Minecraft plugin that transforms tree harvesting into a smooth, realistic experience. Break a single log and watch the entire tree topple over before breaking apart into collectible items.

---

## ✨ Features

### Realistic Tree Felling Animation
- Trees fall over realistically when you break the base log
- Physics-based animation that follows the terrain
- Smooth particle effects enhance the visual experience

### 🪓 Smart Tree Detection
- Automatically detects and removes entire trees
- Works with all vanilla tree types (oak, birch, spruce, jungle, acacia, dark oak, cherry, mangrove, etc.)
- Handles complex tree shapes and branching

### 🎮 Player-Friendly Mechanics
- **Sneak to prevent tree felling** - Want to trim a tree manually? Just hold shift!
- **Player-placed blocks are protected** - Build with logs worry-free! The plugin tracks which blocks you placed
- Tool durability is properly consumed based on tree size
- All drops are natural and collected as usual

### ⚙️ Customizable Animation
- Configure leaf animation behavior
- Adjust animation speed and duration

---

## 📋 Requirements

- **Minecraft Version:** 1.20+ (Paper or compatible fork)
- **Java Version:** 17 or higher
- **Server Software:** PaperMC, Purpur, or any Paper-based server
